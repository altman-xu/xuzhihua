oracle中date与string互换

oracle中date与string互换
Dual伪列介绍

  • 1. date -> string
  • 2. string -> date
  • 转换格式
  • 补充
  • Dual伪列

1. date -> string

to_char(日期,"转换格式") 即把给定的日期按照"转换格式"转换

//显示：08-11-07 13:22:42
select to_char(sysdate,'yy-mm-dd hh24:mi:ss') from dual

2. string -> date

to_date("要转换的字符串","转换的格式")两个参数的格式必须匹配，否则会报错

-- 显示：2005-12-25 13:25:59
select to_date('2005-12-25,13:25:59','yyyy-mm-dd,hh24:mi:ss') from dual 

--则会报错，因为小时hh是12进制，13为非法输入，不能匹配。
select to_date('2005-12-25,13:25:59','yyyy-mm-dd,hh:mi:ss') from dual

转换格式

转换的格式

表示year的：

  1. y 表示年的最后一位
  2. yy 表示年的最后2位
  3. yyy 表示年的最后3位
  4. yyyy 用4位数表示年

表示month的：

  1. mm 用2位数字表示月；
  2. mon 用简写形式 比如11月或者nov ；
  3. month 用全称 比如11月或者november

表示day的：

  1. dd 表示当月第几天；
  2. ddd表示当年第几天；
  3. dy 当周第几天 简写 比如星期五或者fri；
  4. day当周第几天 全写 比如星期五或者friday

表示hour的：

  1. hh 2位数表示小时 12进制；
  2. hh24 2位数表示小时 24小时

表示minute的：mi 2位数表示分钟

表示second的：ss 2位数表示秒 60进制

表示季度的：q 一位数 表示季度 （1-4）

表示周的：

  1. ww 用来表示当年第几周
  2. w 用来表示当月第几周

24小时制下的时间范围：00：00：00-23：59：59
12小时制下的时间范围：1：00：00-12：59：59

补充

当前时间减去7分钟的时间
select sysdate,sysdate - interval '7' MINUTE from dual
当前时间减去7小时的时间
select sysdate - interval '7' hour from dual
当前时间减去7天的时间
select sysdate - interval ’7’ day from dual
当前时间减去7月的时间
select sysdate,sysdate - interval '7' month from dual
当前时间减去7年的时间
select sysdate,sysdate - interval '7' year from dual
时间间隔乘以一个数字
select sysdate,sysdate - 8*interval '7' hour from dual

Dual伪列

含义解释

Dual 是 Oracle中的一个实际存在的表，任何用户均可读取，常用在没有目标表的select语句块中
比如，
select SYSDATE from dual 则返回系统当前的时间：2008-11-07 9:32:49 不同系统可能返回日期的格式不一样
select user from dual 则返回当前连接的用户
select 1+2 from dual 则返回结果：3
