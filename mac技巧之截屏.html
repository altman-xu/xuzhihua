截屏

截屏快捷键
CMD+SHIFT+3 // 截全屏
CMD+SHIFT+4 // 截指定区域
修改截屏保存的文件格式
terminal:

defaults write com.apple.screencapture type png

可以将png改为jpg pdf等
png格式文件占用空间大，jpg的小
