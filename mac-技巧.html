mac 技巧

收集的mac 使用技巧 长期更新

  • 截屏
  • 加密 解密 文件
  • 自动更新
  • 定时熄屏
  • 设置dock自动隐藏 显示出来的延迟时间

截屏

截屏快捷键
CMD+SHIFT+3 // 截全屏
CMD+SHIFT+4 // 截指定区域
修改截屏保存的文件格式
terminal:

defaults write com.apple.screencapture type png

可以将png改为jpg pdf等
png格式文件占用空间大，jpg的小

加密 解密 文件

terminal:

桌面有一份文件 ScreenShotOriginal.png
加密

openssl enc -aes-256-cbc -e -in /Users/xuzhihua/Desktop/ScreenShotOriginal.png  -out /Users/xuzhihua/Desktop/ScreenShotEncrypt.png
// -e encrypt 使用 -aes-256-cbc 加密方式
// 需要输入你想要使用的加密密码

此时会在桌面创建一个 ScreenShotEncrypt.png ，此时可以 ScreenShotOriginal.png 删除
解密

openssl enc -aes-256-cbc -d -in /Users/xuzhihua/Desktop/ScreenShotEncrypt.png  -out /Users/xuzhihua/Desktop/ScreenShotDecrypt.png
// -d decrypt 使用 -aes-256-cbc 加密方式
// 输入加密的密码

此时会在桌面创建一个 ScreenShotDecrypt.png

自动更新

terminal:

defaults write com.apple.com.SoftwareUpdate ScheduleFrequency -int 7
// 默认是7填更新一次，可以改成3天一次或其他的 或 365天一次

定时熄屏

terminal:

caffeinate
// CTRL+C 退出
caffeinate -u -t 7200
// 7200秒后熄屏

设置dock自动隐藏 显示出来的延迟时间

terminal:

defaults write com.app.dock autohide-time-modifier -int 0
// 设置延迟时间为 0（或其他数值）
killall Dock
